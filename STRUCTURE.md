# Noya Nel Web 项目结构说明

## 核心文件

### 1. 主页模块
- `index.html` - 网站主页面
- `index-style.css` - 主页样式
  - 深色主题设计
  - 响应式布局
  - 动画效果
- `index-function.js` - 主页功能
  - 页面导航
  - 状态检查
  - 路由跳转

### 2. 用户中心
- `User/user.html` - 用户中心页面
- `User/user-style.css` - 用户中心样式
  - 信息展示
  - 功能按钮
  - 状态提示
- `User/user-function.js` - 用户功能
  - 信息获取
  - 状态更新
  - 打卡功能

### 3. 通知系统
- `notification.css` - 通知样式
  - 成功/错误样式
  - 动画效果
  - 响应式适配
- `notification.js` - 通知功能
  - 消息显示
  - 自动消失
  - 状态管理
